# CSAT · F2 — Encuesta de satisfacción in-app · Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Encuesta de satisfacción (1-5 estrellas + comentario opcional) que el cliente completa inline en el chat una vez que su ticket pasa a resuelto/cerrado, visible en modo lectura para cliente/técnico/admin.

**Architecture:** App nueva `csat/` con un modelo `CSATResponse` (OneToOne con Ticket), un endpoint simple de submit, y dos campos on-read (`csat`, `can_rate`) inyectados en `TicketSerializer` vía import lazy — mismo patrón de desacople usado por `sla/` en F1 (`tickets_t` no importa `csat` a nivel de módulo). El frontend muestra un prompt de estrellas inline en `ChatPanel` cuando `can_rate` es true, o el resultado en modo lectura cuando `csat` ya existe.

**Tech Stack:** Django 6 + DRF, Vue 3.5.

## Global Constraints

- Escala: **1-5 estrellas** + comentario **opcional**. Una respuesta por ticket, **inmutable** tras enviarla.
- Elegibilidad: `ticket.estado in ("RESOLVED", "CLOSED")`.
- Permiso de submit: sólo `ticket.creado_por` (el dueño). No-dueño → 403. No elegible → 400. Ya calificado → 409. Score fuera de 1-5 → 400.
- Endpoint: `POST /api/csat/<ticket_id>/` (vista simple `APIView`, no ViewSet — un solo verbo).
- `TicketSerializer` suma `csat` (payload o `null`, visible para cualquiera con acceso al ticket) y `can_rate` (bool, `True` sólo para el dueño con ticket elegible y sin respuesta previa). Import lazy de `csat` desde `tickets_t/serializers.py` — `tickets_t` NO importa `csat` a nivel de módulo.
- **Sin notificación dedicada**: se apoya en la notificación existente de "ticket resuelto" (Fase 2) como nudge. Sin email propio (in-app únicamente).
- Estética v2: hairline 0.5px, CSS vars, voseo. Frontend sin test runner: verificación = `npm run build` limpio.
- Tests backend: `django.test.TestCase` + `APIClient` + `force_authenticate`, siguiendo el patrón de `backend/sla/tests.py` / `backend/tickets_t/tests.py`. Correr desde `backend/`: `python manage.py test csat -v 2` (agregar `--keepdb` si "database exists"). System Python `C:\Python312\python.exe`; no venv.

`backend/csat/views.py`:
```python
from rest_framework.permissions import IsAuthenticated
from rest_framework.response import Response
from rest_framework.views import APIView

from tickets_t.models import Ticket

from .eligibility import is_eligible
from .models import CSATResponse
from .payloads import csat_payload
from .serializers import CSATSubmitSerializer


class SubmitCsatView(APIView):
    permission_classes = [IsAuthenticated]

    def post(self, request, ticket_id):
        ticket = Ticket.objects.filter(pk=ticket_id).first()
        if ticket is None:
            return Response({"detail": "No encontrado."}, status=404)
        if ticket.creado_por_id != request.user.id:
            return Response({"detail": "Solo el creador del ticket puede calificar."}, status=403)
        if not is_eligible(ticket):
            return Response({"detail": "El ticket todavía no está resuelto."}, status=400)
        if getattr(ticket, "csat", None) is not None:
            return Response({"detail": "Ya calificaste este ticket."}, status=409)

        ser = CSATSubmitSerializer(data=request.data)
        ser.is_valid(raise_exception=True)
        CSATResponse.objects.create(ticket=ticket, **ser.validated_data)
        return Response(csat_payload(ticket), status=201)
```

<script setup>
import { ref } from "vue";
import { submitCsat } from "../../api/csat.api";

const props = defineProps({ ticketId: { type: Number, required: true } });
const emit = defineEmits(["submitted"]);

const score = ref(0);
const hoverScore = ref(0);
const comment = ref("");
const submitting = ref(false);
const error = ref("");

async function submit() {
  if (!score.value) return;
  submitting.value = true;
  error.value = "";
  try {
    const payload = await submitCsat(props.ticketId, { score: score.value, comment: comment.value.trim() });
    emit("submitted", payload);
  } catch (err) {
    error.value = err?.response?.data?.detail || "No se pudo enviar la calificación.";
  } finally {
    submitting.value = false;
  }
}
</script>
